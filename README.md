# Youtube_comments_sentiment_analysis

This project is machine learning based project.
In this project you have to insert the url of desired youtube video in to the google sheets cell which is connected to the API that helps to scrap the viedo's comments and saved in the excel sheet format.
After that you have to save the excel sheet in the respective compiler you are running the code in and the algorithm will provide you the visual representation of sentiments stated in the comment section of the respective youtube video.
